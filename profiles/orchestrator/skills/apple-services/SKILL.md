# Apple Services

Apple ecosystem integrations. All require macOS.

## Section A: Apple Notes

Read and search Apple Notes on macOS. Use the AppleScript/JXA bridge to access the Notes app.

## Section B: Apple Reminders

Create, read, and manage Apple Reminders. Use AppleScript to interact with the Reminders app.

## Section C: iMessage

Send and receive iMessages. Uses the Messages app on macOS via AppleScript.

## Section D: Find My

Locate Apple devices via the Find My service. Returns device location data.
