# Research Workflow

## Section A: arXiv Paper Search (arxiv)

Search and retrieve academic papers from arXiv by keyword, author, category, or ID.

- **Search by keyword:** `search_files` equivalent for arXiv API
- **Browse by category:** cs.AI, cs.LG, cs.CL, etc.
- **Get paper by ID:** `2103.12345` format
- **Download PDF:** via arXiv URL patterns

## Section B: Blog/RSS Monitoring (blogwatcher)

Monitor blogs and RSS/Atom feeds for new content.

- Add feeds to watch list
- Check for new posts since last check
- Summarize new content

## Section C: LLM Wiki (llm-wiki)

Karpathy's LLM Wiki — build and query an interlinked markdown knowledge base.

- Build: ingest documents, create wiki links
- Query: find related topics, follow links
- Use for personal knowledge management

## Section D: Polymarket (polymarket)

Query Polymarket prediction market data: markets, prices, orderbooks, history.

- Get market prices and odds
- Query orderbooks
- Historical price data

## Section E: Research Paper Writing (research-paper-writing)

Write academic research papers with proper structure: abstract, introduction, methodology, results, discussion, conclusion.

- Follow academic conventions
- Cite sources properly
- Structure for peer review
