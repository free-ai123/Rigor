# Web Design

## Section A: Sketch (sketch)

Quick throwaway HTML mockups. Generate 2-3 design variants to compare before committing to a final design.

- Create simple HTML/CSS prototypes
- Explore layout, color, and typography options
- Compare variants side by side
- Bias toward speed — these are disposable

## Section B: Claude Design (claude-design)

Design one-off HTML artifacts: landing pages, presentation decks, interactive prototypes.

- Higher fidelity than sketch — meant to be closer to final
- Landing pages, decks, form prototypes, interactive demos
- Single self-contained HTML file when possible
- Use real design systems, not generic templates

## Section C: Popular Web Designs (popular-web-designs)

Reference library of 54 real design systems (Stripe, Linear, Vercel, etc.) as HTML/CSS.

- Use as reference when building web artifacts
- Match specific design systems to user needs
- Learn patterns from production sites
- Provides concrete examples of typography, spacing, color, and component patterns
